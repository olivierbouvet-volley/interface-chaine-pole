💡 Calibration depuis n'importe quel point

  1. Mettez la vidéo en pause exactement sur l'action voulue
    (ex : le service à 11-10 si votre vidéo démarre là)
  2. Cliquez sur cette action dans la colonne de droite →
  3. L'offset se calcule automatiquement : toutes les actions suivantes sont recalées
  4. Ajustez avec ±0.1s si besoin
  5. 💾 Sauvegardez quand la synchro est bonne
ℹ️ Les actions sans timestamp (saisies manuellement avant l'enregistrement) affichent —. Elles ne servent pas à calibrer — cliquez sur la première action avec un temps vidéo.
L'offset sera sauvegardé dans Firebase et utilisé pour synchroniser les actions DVW avec la vidéo.
